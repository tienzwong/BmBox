# BmBox ERP — บริษัท เบลสโมทีฟ จำกัด

ระบบ ERP ใช้งานภายในสำหรับโรงพิมพ์แพคเกจจิ้งกระดาษ (แบรนด์ **BmBox**)
เริ่มจากโมดูล **ใบเสนอราคา** ที่คำนวณการวางชิ้นงานบนแผ่นกระดาษ (imposition) ให้ใช้กระดาษคุ้มที่สุดโดยอัตโนมัติ

## สิทธิ์การใช้งาน (Login + RBAC)

ระบบล็อกอินด้วย session cookie ป้องกันการเข้าถึง และ**คุมการมองเห็นต้นทุน/ราคาตามแผนก**

| บทบาท | เห็นต้นทุน/กำไร | เห็นราคาขาย | สร้างใบเสนอราคา | จัดการข้อมูลหลัก | จัดการผู้ใช้ |
|-------|:---:|:---:|:---:|:---:|:---:|
| ผู้ดูแลระบบ (admin) | ✓ | ✓ | ✓ | ✓ | ✓ |
| ผู้บริหาร (management) | ✓ | ✓ | ✓ | ✓ | |
| ฝ่ายบัญชี (accounting) | ✓ | ✓ | | | |
| ฝ่ายขาย (sales) | | ✓ | ✓ | | |
| ฝ่ายผลิต (production) | | | | | |

ฝ่ายผลิตเห็นเฉพาะ**สเปกงาน** (ขนาด/จำนวนชิ้นต่อแผ่น/จำนวนแผ่น/เครื่องพิมพ์) เพื่อใช้ผลิต แต่ไม่เห็นต้นทุนและราคา — เอกสารจะแสดงเป็น "ใบสั่งงานผลิต" แทน "ใบเสนอราคา"

บัญชีเริ่มต้น (รหัสผ่าน = ชื่อผู้ใช้ + `123` เช่น `admin` / `admin123`): `admin`, `manager`, `sales`, `account`, `prod` — **เปลี่ยนรหัสผ่านก่อนใช้งานจริง**

## ฟีเจอร์หลัก

- **คำนวณการตัดกระดาษอัตโนมัติ** — หาจำนวนชิ้นต่อแผ่น (ups) สูงสุด โดยลองทั้งวางตรง / หมุน 90° / ตัดผสม (dutch cut) แล้วเลือกแบบที่คุ้มที่สุด พร้อมแสดง **แผนผังการวางบนแผ่นพิมพ์** เป็นภาพ
- **คิดต้นทุนครบ** — ค่ากระดาษ (คำนวณราคา/แผ่นจากน้ำหนัก × ราคา/กก.), ค่าเพลท, ค่าพิมพ์ต่อสี, เผื่อเสีย, ค่าเซ็ตเครื่อง, บวกกำไร
- **ออกใบเสนอราคา** — เลขที่อัตโนมัติ (QT-ปปดด-####), หลายรายการต่อใบ, VAT/ส่วนลด, พิมพ์/บันทึก PDF
- **คลังกระดาษ & ลูกค้า** — จัดการชนิดกระดาษ ขนาดแผ่น ราคา และฐานข้อมูลลูกค้า

## เทคโนโลยี

- Next.js 16 (App Router) + React 19 + TypeScript
- Tailwind CSS v4 — โทนขาว 70% / น้ำเงิน 30%
- Prisma 6 + SQLite

## เริ่มใช้งาน

```bash
npm install          # ติดตั้ง + prisma generate อัตโนมัติ
npm run db:push      # สร้างตารางในฐานข้อมูล
npm run db:seed      # ใส่ข้อมูลกระดาษ/ลูกค้าตัวอย่าง
npm run dev          # เปิดที่ http://localhost:3000
```

## โครงสร้างสำคัญ

| ไฟล์ | หน้าที่ |
|------|---------|
| `src/lib/imposition.ts` | เครื่องคำนวณการวางชิ้นบนแผ่น (หา ups สูงสุด) |
| `src/lib/cost.ts` | โมเดลคิดต้นทุน/ราคา |
| `src/components/QuotationForm.tsx` | หน้าสร้างใบเสนอราคาพร้อมคำนวณสด |
| `src/components/SheetView.tsx` | วาดแผนผังการวางบนแผ่นกระดาษ (SVG) |
| `prisma/schema.prisma` | โครงสร้างฐานข้อมูล |
